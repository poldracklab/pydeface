# PyDeface
A tool to remove facial structure from MRI images.

## Installation
```
pip install pydeface
```
or
```
git clone https://github.com/poldracklab/pydeface.git
cd pydeface
pip install .
```

## How to use
```
pydeface infile.nii.gz
```

Also see the help for additional options:
```
pydeface --help
```
