# Changelog

## v2.1.0 - 8 May 2026

New feature release in the 2.1.x series.

This release has tested support for Python 3.10-3.14, and
has dropped support for Python 3.9 and below.

* Start keeping a changelog.
* Use pixi to maintain a lockfile and build environments.
* Update Docker container to wrap a pixi environment.
* Update build backend from setuptools to flit.
* Add pre-commit hooks to enforce consistent style.
